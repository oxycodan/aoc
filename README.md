# aoc2023

## Advent of Code 2023 | TypeScript Answers

### How to run

```bash
npm install
npm run build
npm start
```

### User Input
`./public/` contains user input such as for day one: `./public/dayOne.txt`.

### Example Output
```
> aoc2023@1.0.0 start 
> node dist/index.js
```
